# Snake Game 🎮

## Overview
This repository contains a simple yet classic **Snake Game** built in Python using the `pygame` library.  
It demonstrates core game development concepts such as event handling, collision detection, score tracking, and rendering graphics.

## Features
- Arrow key controls for snake movement
- Randomly generated food items
- Real-time score display
- Game-over screen with replay option
